## Getting started

#### 1) Install dependencies

```
npm i
```

#### 2) Start dev server

```
npm start
```

You will see 400 when you access to http://127.0.0.1:3000/ because LocalStrategy blocks the request.


##### 3) Run tests

```
npm test
```

It passes because it uses mock strategy.
